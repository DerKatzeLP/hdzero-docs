# HDZero ECO 摄像头（已过时）

## 描述

HDZero ECO 摄像头旨在通过提供一种低成本、轻重量且耐用的模拟视频替代方案，让更多飞手能够体验数字 FPV。数字视频信号具有稳定的色彩还原和 720p 逐行扫描视频，并具备 HDZero 一贯的低延迟特性。

HDZero ECO 摄像头经过精心设计，非常适合像 Tiny Whoops 这样的超轻型无人机，每一克重量都至关重要。摄像头高达 98 度的垂直视场角非常适合快速的室内贴近飞行。
