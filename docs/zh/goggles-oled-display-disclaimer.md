# OLED 显示屏免责声明

1. 由于 OLED 技术的特性，制造过程中可能会出现坏点，这属于正常现象。请参考以下标准：

    - **单个坏点**：属于正常情况，不影响产品性能。

    - **两个连续坏点**（任意方向——水平、垂直或对角线）：可接受，但总数不得超过 **4 处**。

    - **三个或以上连续坏点**（任意方向）：属于 **异常情况**，可能符合保修评估或更换条件。

2. OLED 显示屏相较传统 LCD 面板可呈现更鲜艳的色彩，但在长时间显示相同内容时，可能会出现“图像残留”或“烧屏”等问题。
