# HDZero Nano V2 摄像头

## 描述

Runcam Nano V2 HDZero 摄像头是一款高清 1280x720p60 摄像头，专为 HDZero 图传设计。它支持摄像头菜单，可调节亮度、对比度等设置，在低光环境下表现出色。

## 规格

| 参数          | 值                       |
| ------------- | ------------------------ |
| 图像传感器    | 1/2"                     |
| 接口          | MIPI                     |
| 分辨率        | 1280x720@60fps           |
| 最大视场角    | D:155° H:122° V:89°(4:3) |
| 快门          | 滚动快门                 |
| 净重          | 4.3g                     |
| 尺寸          | 长14mm * 宽16mm * 高21mm |
